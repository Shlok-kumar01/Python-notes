# Programming
- Programming is the process of giving step-by-step instructions to a computer so it can perform a specific task.

- If we skip or change the step then result will be different.
- Real life Example : 
	Think of making tea:
	1. Boil water
	2. Add tea leaves
	3. Add milk and sugar
	4. Stir
	5. Serve
## Programming Language
- A programming language is a language used to write instructions that a computer can understand.

 - There are two types of programming language
	1. **Low-Level :** close to computer's hardware & hard to understand for humans
		Example: machine language(0s and 1s), Assembly Language
	2. **High-Level :** close to human language & easy to read, write and understand
		Example: C, C++, Java, Python

---
# Python 
- It is a high level programming language.
- Created by ***Guido van Rossum*** in 1991.

- Python is an interpreted language.
	- An **interpreted language** is a programming language in which code is executed by an interpreter line by line at runtime instead of being compiled into machine code beforehand.

- Python extension→ **.py**
- Used in Web development, Cybersecurity, AI & ML, Automation, Data Science, Game Development etc.
